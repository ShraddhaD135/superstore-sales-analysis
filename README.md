Project Overview

This project performs end-to-end exploratory data analysis (EDA) on a retail Superstore dataset using Python. The objective is to analyze sales performance, profitability, customer segments, regional trends, and top-performing products to generate actionable business insights.


Business Objective

Identify top revenue-generating categories
Determine the most profitable customer segment
Analyze regional sales distribution
Track monthly sales trends
Identify top-performing products


Tech Stack

Python
Pandas
Matplotlib
VS Code


Dataset Description

The dataset includes:
Order Date
Sales
Profit
Category
Region
Segment
Product Name


Key Findings

Total Sales: 12,642,905
Total Profit: 1,467,457
Highest Revenue Category: Technology
Most Profitable Category: Technology
Highest Profit Segment: Consumer
Top Sales Region: Central


Key Business Insights

Technology is the core revenue and profit driver.
Consumer segment contributes the highest profit.
Central region generates maximum sales.
Furniture shows high sales but comparatively lower profit margins.


How to Run the Project

Install required libraries:
pip install pandas matplotlib
Run:
python sales_analysis.py
